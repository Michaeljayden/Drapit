## Afkortingen
| Term | Betekenis | Context |
|------|-----------|---------|
| VTON | Virtual Try-On | De AI-kernfunctie van Drapit |
| prod | Productie | Drapit.io live omgeving |
| dev | Development | Lokale of staging omgeving |
| PR | Pull Request | GitHub code review |
